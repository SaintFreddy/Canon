### 11. LAYER 6 — MEMORY SYSTEM
Memory must be modular and typed.

The engine should support multiple memory classes rather than one undifferentiated store.

#### 11.1 Working memory

Short-lived execution-local state.

Features:
- intermediate conclusions
- open questions
- temporary hypotheses
- local plans
- sub-result caching
- ephemeral relational state

#### 11.2 Episodic memory

Memory of prior executions and what happened.

Features:
- execution summaries
- outcome summaries
- failure patterns
- prior decisions
- prior tool usage traces
- prior validation results

#### 11.3 Semantic memory

Memory of facts, entities, relationships, constraints, and normalized statements.

Features:
- entity records
- relation graphs
- fact objects
- structured constraints
- semantic retrieval
- contradiction tracking
- supersession handling

#### 11.4 Procedural memory

Memory of how to do things.
